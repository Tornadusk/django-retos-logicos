3. SECCIONES DETALLADAS
================================================================================

🔐 AUTHENTICATION AND AUTHORIZATION
------------------------------------
Esta sección maneja la autenticación básica de Django.

📁 Groups (Grupos):
   • Propósito: Crear grupos de usuarios con permisos específicos
   • Ejemplos de grupos:
     - "Profesores": Pueden crear y editar retos
     - "Estudiantes": Solo pueden resolver retos
     - "Moderadores": Pueden gestionar usuarios
   • Acciones disponibles:
     - "+ Add": Crear nuevo grupo
     - "Change": Editar grupos existentes

👥 Users (Usuarios):
   • Propósito: Gestionar todos los usuarios del sistema
   • Información que contiene:
     - Nombre de usuario, email, nombre, apellido
     - Estado activo/inactivo
     - Permisos de staff y superusuario
     - Fecha de registro y último login
   • Acciones disponibles:
     - "+ Add": Crear nuevo usuario
     - "Change": Editar usuarios existentes

================================================================================
👤 CUENTAS (App de usuarios)
-----------------------------
Esta sección maneja los perfiles extendidos de los usuarios.

📊 Perfiles de Usuario:
   • Propósito: Perfiles extendidos con estadísticas de juego
   • Información que contiene:
     - Usuario asociado
     - Puntuación total acumulada
     - Número de retos completados
     - Nivel del usuario
     - Fecha de registro
   • Se crea automáticamente cuando se registra un nuevo usuario
   • Acciones disponibles:
     - "+ Add": Crear perfil manualmente (raro)
     - "Change": Editar perfiles existentes

================================================================================
🎮 JUEGO (App de lógica de juego)
----------------------------------
Esta sección maneja la lógica del juego y las puntuaciones.

🎯 Intentos:
   • Propósito: Registra todas las respuestas de usuarios a retos
   • Información que contiene:
     - Usuario que hizo el intento
     - Reto que intentó resolver
     - Respuesta proporcionada por el usuario
     - Si la respuesta es correcta o incorrecta
     - Puntuación obtenida
     - Fecha y hora del intento
     - Tiempo de respuesta (si se registra)
   • Acciones disponibles:
     - "+ Add": Crear intento manualmente (raro)
     - "Change": Editar intentos existentes

🏆 Rankings:
   • Propósito: Clasificación de usuarios por puntuación
   • Información que contiene:
     - Posición en el ranking
     - Usuario
     - Puntuación total
     - Número de retos completados
     - Fecha de última actualización
   • IMPORTANTE: Solo visualización (no se puede editar)
   • Se actualiza automáticamente cuando los usuarios completan retos
   • Acción disponible:
     - "View": Solo visualizar (no se puede modificar)

================================================================================
🧩 RETOS (App principal)
-------------------------
Esta es la sección más importante, maneja los retos y categorías.

📂 Categorías:
   • Propósito: Organizar los retos por temas o áreas
   • Información que contiene:
     - Nombre de la categoría
     - Descripción
     - Color (en formato hexadecimal) para identificación visual
   • Ejemplos de categorías:
     - "Lógica Básica" (#28a745 - verde)
     - "Matemáticas" (#007bff - azul)
     - "Secuencias" (#ffc107 - amarillo)
     - "Geometría" (#dc3545 - rojo)
   • Acciones disponibles:
     - "+ Add": Crear nueva categoría
     - "Change": Editar categorías existentes

🎯 Retos:
   • Propósito: Los acertijos y problemas que resuelven los usuarios
   • Información que contiene:
     - Título del reto
     - Descripción breve
     - Enunciado completo del problema
     - Respuesta correcta
     - Explicación de la solución
     - Categoría a la que pertenece
     - Nivel de dificultad (Fácil, Medio, Difícil, Experto)
     - Puntos que otorga
     - Estado activo/inactivo
     - Estadísticas: intentos totales y éxitos
     - Metadatos: fecha de creación, creado por
   • Acciones disponibles:
     - "+ Add": Crear nuevo reto
     - "Change": Editar retos existentes

4. FUNCIONALIDADES POR SECCIÓN
================================================================================

🆕 CREAR NUEVO RETO
-------------------
1. Ir a RETOS → Retos → "+ Add"
2. Completar los campos:
   • Título: Nombre descriptivo del reto
   • Descripción: Resumen breve
   • Enunciado: Problema completo
   • Respuesta correcta: Solución exacta
   • Explicación: Cómo se resuelve
   • Categoría: Seleccionar de la lista
   • Dificultad: Fácil (10 pts), Medio (20 pts), Difícil (50 pts), Experto (100 pts)
   • Puntos: Se ajusta automáticamente según dificultad
   • Activo: Marcar para que esté disponible
3. Guardar

📊 MONITOREAR ESTADÍSTICAS
---------------------------
1. Ver intentos: JUEGO → Intentos → "Change"
   - Filtrar por usuario, reto, fecha
   - Ver respuestas correctas/incorrectas
   - Analizar patrones de dificultad

2. Ver rankings: JUEGO → Rankings → "View"
   - Top usuarios por puntuación
   - Progreso de usuarios
   - Identificar usuarios más activos

3. Ver perfiles: CUENTAS → Perfiles de Usuario → "Change"
   - Puntuaciones individuales
   - Retos completados por usuario
   - Progreso de nivel

👥 GESTIONAR USUARIOS
---------------------
1. Ver usuarios: AUTHENTICATION → Users → "Change"
   - Estado activo/inactivo
   - Información de contacto
   - Permisos de acceso

2. Ver perfiles: CUENTAS → Perfiles de Usuario → "Change"
   - Estadísticas de juego
   - Progreso individual

🎯 PASO 2: CREAR RETOS ESTRUCTURADOS
================================================================================

🎯 OBJETIVO: Crear retos bien organizados y educativos

📝 PROCESO:
1. **Ir a RETOS → Retos → "Change"**
2. **Revisar retos existentes** para entender la estructura
3. **Crear nuevo reto:**
   • Click en "+ Add"
   • Completar campos obligatorios:

📋 CAMPOS OBLIGATORIOS:
   • **Título**: Nombre descriptivo y atractivo
   • **Descripción**: Resumen breve (1-2 líneas)
   • **Enunciado**: Problema completo y claro

   • **Dificultad**: Fácil/Medio/Difícil/Experto

📋 CAMPOS OPCIONALES:
   • **Puntos**: Se ajusta automáticamente según dificultad
   • **Creado por**: Se asigna automáticamente

4. **Guardar el reto**

✅ RESULTADO: Retos bien estructurados y educativos

7. SOLUCIÓN DE PROBLEMAS
================================================================================

🚨 PROBLEMAS COMUNES
---------------------

**Problema**: Usuario no puede acceder
**Solución**: Verificar en Users que esté activo

**Problema**: Reto no aparece en la lista
**Solución**: Verificar que esté marcado como "Activo"

**Problema**: Ranking no se actualiza
**Solución**: Los rankings se actualizan automáticamente, verificar intentos

**Problema**: Estadísticas incorrectas
**Solución**: Revisar intentos y recalcular si es necesario

**Problema**: No se pueden crear retos
**Solución**: Verificar permisos de usuario y categorías existentes
